Place real S3 test profiles in this folder.

Current TM2500 testboard profiles:
- TEST_EQP.s3n: EQP controller base test.
- TEST_UD10.s3n: UD10 / gas detector test.
- TEST_IR.s3n: IR gas detector test.
- TEST_EDIO.s3n: EDIO module, hardwired I/O, horns, strobes, and release LED test.

Not used by TM2500 XPRESS:
- TEST_FMEME_DET.s3n: reserved for a future package with flame detectors.
- TEST_DCIO.s3n: reserved for a future package with DCIO modules.

The HMI Real Test Profile panel uses these filenames when an addressable device is switched to REAL mode.
