# EQP HMI Visual Prototype

Open `index.html` in a browser to view the first static HMI/SCADA concept.

Recommended viewing:

- Full screen
- 16:9 monitor or TV
- 1920 x 1080 or larger

Included views:

- Package overview
- Real turbine Fire & Gas layout reference
- Drawing reference
- Loop topology
- Alarm banner
- Device detail panel
- Virtual control panel
- Event history
- Bottom SCADA navigation

This prototype is visual only. It does not include live Modbus, database storage, or real device communication yet.
